05_HTML_태그소개_텍스트-서식

특정 단어 진하게 표시하기 : b태그

텍스트를 강조할 때 사용하는 방법 중 1가지(bold 사용)

특정 단어 진하게 표시하기 : strong태그

텍스트를 강조할 때 사용하는 방법 중 2번째(strong 강조 사용)

텍스트 기울이기 : italic태그

텍스트의 특정 부분을 기울여서 표시하기 (텍스트 기울이기)

텍스트의 특정 부분은 작게 표시하기 : small 태그

텍스트의 특정 부분만 작게 표시하고자 할 때 사용 (smail)

텍스트의 삭제 표현하기 : del 태그 (판매 사이트에서 주로 사용됨)

텍스트에서 삭제된 텍스트를 표현하고자 할 때 사용 (2,500 할인가격 : 2,000)

텍스트에 추가(삽입)된 표현하기 : ins 태그

텍스트에서 추가된 부분을 표시할 때 사용되며, 언더라인(밑줄)이 표시됨 (추가되었습니다.)

텍스트에 윗첨자(sup태그) / 아래첨자(sub) 사용

첨자에는 두가지가 있습니다.
- 하나는 윗첨자(sup) 가 있습니다.
- 다른 하나는 아래첨자(sub) 가 있음

쌍따옴표 넣기 : q태그

Html 태그 내에 문장 중에 특정 부분을 쌍따옴표로 처리하고자 할 때
쌍따옴표는 이렇게 사용

블록쿼터 영역 만들기 : blockquote 태그

블록쿼터란 특정 영역을 블록처럼 별도로 표현하고자 할 때 사용

이곳은 블록(blockquote)으로 별도 지정된 영역입니다.

단축영단어의 풀네임을 별도로 표시하고자 할 때 : abbr 태그

단축 영단어의 WHO라는 단어의 Full Name을
보여주기 보다는 WHO 단어를 addr 태그로 감싸서 설명을 달아줍니다.
* 밑줄로 WHO가 표시됩니다.
