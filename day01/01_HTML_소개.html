* 브라우저에는 body영역 내에 있는 내용들만 보이게 됨
* HTML은 웹 문서를 기술하기 위한 마크업 언어(태그 언어)
* 웹 페이지를 통칭해서 -> HTML 문서라고 칭함
* HTML 문서 내에는 : HTML태그 + CSS + Javascript 로 구성되어 있음
* VS-Code에서 HTML 문서를 브라우저에서 바로 보여질 수 있도록 하는 확장플러그이 있음
* 확장플러그인 설치 : Open Browser Preview (실행은 Ctrl + F1)
* 웹페이지의 수정이 있는 경우에는 -> 브라우저에서 새로고침으로 확인 가능(이미 열려있는 경우)
